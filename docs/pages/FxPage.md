# FxPage (환율 현황)

**파일**: `src/pages/FxPage.tsx`  
**라우트**: `/fx`, `/fx/:currency`  
**데이터**: GAS API (환율) + `daily` 테이블 (외화 잔고)  
**권한**: 전 역할 읽기 전용

---

## 환율 데이터 소스

```
GAS Web App → 네이버 금융 (하나은행 기준)
통화: USD / EUR / JPY(100단위) / GBP / CNY
```
→ `VITE_GAS_API_URL` 미설정 시 경고 배너 표시, 환율 0원

## 화면 구성

### 환율 카드 (5개)
- 국기 이모지 + 통화코드 + 단위 + 현재 환율
- 보유잔고 있는 경우: 잔고 금액 + 원화환산
- 클릭 시 파란 테두리 하이라이트

### 외화 잔고 현황 테이블
- 데이터 소스: `daily` 테이블 최신 1건의 fx_* 컬럼
- 보유잔고 있는 통화만 표시
- 비중 프로그레스 바 (전체 fx_krw 대비)
- 합계 행: `fx_krw` 표시

### 환산 계산기
- 외화 → 원화 / 원화 → 외화 양방향
- 통화 선택 → 실시간 환산

## 딥링크

```
/fx/USD → USD 카드 파란 테두리 하이라이트 + 계산기 USD 선택
```

## 변경 포인트

- **통화 추가**: `FX_META` 객체 + `FX_CODES` 배열 + GAS 코드 수정 + DB fx_* 컬럼 추가
- **환율 소스 변경**: `useFx.ts`의 `fetchRates()` 함수 수정
- **JPY 단위 변경**: `FX_UNITS.JPY = 100` (현재) → 1로 변경 시 표시 방식도 수정
