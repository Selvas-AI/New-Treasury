# FxPage (환율 현황)

**파일**: `src/pages/FxPage.tsx`  
**라우트**: `/fx`, `/fx/:currency`  
**데이터**: GAS API (환율) + `daily` 테이블 (외화 잔고)  
**권한**: 읽기 전용

> ⚠️ **이관 안내 (2026-06-05)**: FX 정책관리 탭(Target Band, 변동폭 계산, 적정 외화보유한도)은  
> **자금정책 관리 (`/policy`) → 💱 FX 정책 탭**으로 이관됨.

---

## 화면 구성 (단일 뷰)

### 환율 카드 (5개)
- 국기 이모지 + 통화코드 + 단위 + 현재 환율
- 보유잔고 있는 경우: 잔고 금액 + 원화환산 표시
- 클릭 시 파란 테두리 하이라이트 (딥링크 연동)

### 외화 잔고 현황 테이블
- 데이터 소스: `daily` 테이블 최신 1건의 `fx_*` 컬럼
- 보유잔고 있는 통화만 표시
- 비중 프로그레스 바 (전체 `fx_krw` 대비)
- 합계 행: `fx_krw` 표시

### 환산 계산기
- 외화 → 원화 / 원화 → 외화 양방향 전환
- 통화 선택 → 실시간 환산 결과

---

## 환율 데이터 소스

```
GAS Web App → 네이버 금융 (하나은행 기준)
통화: USD / EUR / JPY(100단위) / GBP / CNY
```
→ `VITE_GAS_API_URL` 미설정 시 경고 배너 표시, 환율 0원

---

## 딥링크

```
/fx/USD → USD 카드 파란 테두리 하이라이트 + 계산기 USD 선택
```
