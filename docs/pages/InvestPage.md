# InvestPage (운용자금)

**파일**: `src/pages/InvestPage.tsx`  
**라우트**: `/invest`, `/invest/:company`, `/invest/:company/:id`  
**데이터**: `investments` 테이블 (`product !== '국채'`)  
**권한**: master·company 편집 / ceo 읽기 전용

> ⚠️ **이관 안내 (2026-06-05)**: FVPL 리스크 탭(국채 Duration·금리 시나리오)은  
> **자금정책 관리 (`/policy`) → 📈 FVPL 리스크 탭**으로 이관됨.

---

## 탭 구성

| 탭 | 필터 | 설명 |
|----|------|------|
| 운용 중 | `active = true` | 현재 운용 중인 자금 |
| 만기·종료 | `active = false` | 만기 처리된 이력 |
| ~~FVPL 리스크~~ | ~~국채 최신 1건~~ | **PolicyPage로 이관됨** |

---

## KPI 요약 (상단 3개 카드)

| 카드 | 계산 |
|------|------|
| 가용 합계 | `active=true, available='가용'` 합계 |
| 불가용 합계 | `active=true, available='불가용'` 합계 |
| 총 운용 합계 | 가용 + 불가용 |

---

## 이관 안내 배너

페이지 상단에 상시 표시:
```
📊 FVPL 리스크 분석은 자금정책 관리 → FVPL 리스크 탭에서 확인하세요.
```

## 등록 필드

| 필드 | 필수 | 설명 |
|------|------|------|
| 금융기관 | ✓ | 예: KB국민은행 |
| 상품유형 | ✓ | 정기예금/RP/MMF/발행어음/CMA/채권/기타 |
| 통화 | | KRW/USD/EUR/JPY/GBP/CNY |
| 금액 | ✓ | 원화 기준 |
| 수익률(%) | | |
| 가용 여부 | | 가용/불가용 |
| 시작일 | | YYYY-MM-DD |
| 만기일 | ✓ | YYYY-MM-DD |
| 취득가액 | | 수익률 계산용 |

### 수익률 미리보기
`금액 > 0 && 취득가액 > 0` 이면 입력 폼 하단에 실시간 표시

---

## D-day 색상

| 기간 | 색상 | 행 배경 |
|------|------|---------|
| D-7 이하 | 빨강 | 빨간 배경 (`bg-red-50`) |
| D-30 이하 | 주황 | 주황 배경 |
| D-30 초과 | 회색 | 기본 |

---

## 주요 액션

- **만기**: `setActive(id, false)` → 만기·종료 탭으로 이동
- **복원**: `setActive(id, true)` → 운용 중 탭으로 복귀
- **딥링크**: `/invest/:company/:id` → 해당 레코드 수정 폼 자동 열기
