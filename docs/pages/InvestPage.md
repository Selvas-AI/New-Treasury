# InvestPage (운용자금)

**파일**: `src/pages/InvestPage.tsx`  
**라우트**: `/invest`, `/invest/:company`, `/invest/:company/:id`  
**데이터**: `investments` 테이블 (`product !== '국채'`)  
**권한**: master·company 편집 / ceo 읽기 전용

---

## 탭 구성

| 탭 | 필터 | 설명 |
|----|------|------|
| 운용 중 | `active = true` | 현재 운용 중인 자금 |
| 만기·종료 | `active = false` | 만기 처리된 이력 |

## 등록 필드

| 필드 | 설명 |
|------|------|
| 금융기관 | 예: KB국민은행 |
| 상품유형 | 정기예금/RP/MMF/발행어음/CMA/채권/기타 |
| 통화 | KRW/USD/EUR/JPY/GBP/CNY |
| 금액 | 원화 기준 |
| 수익률(%) | |
| 가용 여부 | 가용/불가용 |
| 시작일 / 만기일 | |
| 취득가액 | 수익률 계산용 |

## D-day 색상

| 기간 | 색상 | 행 배경 |
|------|------|---------|
| D-7 이하 | 빨강 | 빨간 배경 |
| D-30 이하 | 주황 | 주황 배경 |
| D-30 초과 | 회색 | 기본 |

## 주요 액션

- **만기**: `setActive(id, false)` → 만기·종료 탭으로 이동
- **복원**: `setActive(id, true)` → 운용 중 탭으로 복귀
- **취득가액 입력 시**: 등록 폼에서 수익률 미리보기 표시

## 변경 포인트

- **상품유형 추가**: `PRODUCT_OPTIONS` 배열에 항목 추가
- **D-day 임계값**: `dday <= 30`, `dday <= 7` 조건 수정
