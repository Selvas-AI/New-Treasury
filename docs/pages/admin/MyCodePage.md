# MyCodePage (코드 변경)

**파일**: `src/pages/admin/MyCodePage.tsx`  
**라우트**: `/admin/mycode`  
**권한**: 전 역할 (자신의 코드만 변경)

---

## 동작 흐름

```
현재 계정 정보 표시 (이름, 역할)
↓
새 접근 코드 입력 (6자 이상)
새 코드 확인 (일치 검증)
↓
저장 → Supabase access_codes 업데이트
↓
성공 메시지 2초 표시 → 자동 로그아웃 → 재로그인 유도
```

## 유효성 검사

| 조건 | 오류 메시지 |
|------|-----------|
| 6자 미만 | "코드는 6자 이상이어야 합니다." |
| 불일치 | "새 코드가 일치하지 않습니다." |
| 현재 코드와 동일 | "현재 코드와 동일합니다." |

## 보안 참고

- 코드 변경 후 `sessionStorage` 초기화 → 재로그인 필요
- `access_codes` 테이블은 RLS(Row Level Security) 미적용 상태
  → Supabase 대시보드에서 RLS 설정 권장

## 변경 포인트

- **최소 길이**: `newCode.length < 6` 조건 수정
- **로그아웃 지연**: `setTimeout 2000` 수정
