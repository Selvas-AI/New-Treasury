# HistoryPage (자금 변동 이력)

**파일**: `src/pages/HistoryPage.tsx`  
**라우트**: `/history`, `/history/:company`, `/history/:company/:from/:to`  
**데이터**: `daily` + `investments` + `loans` 테이블  
**권한**: 전 역할 읽기 전용

---

## 화면 구성

```
[기간 탭: 7일 | 30일 | 90일 | 1년]  [📋 표 | 📈 차트]

[KPI: 최근 운전자금 | 최근 운용자금 | 차입금 | 순현금 (최고/최저)]

[표 뷰 또는 차트 뷰]
```

## 날짜별 계산

```
운전자금 = krw_demand + krw_govt + krw_mmda + fx_krw
운용자금 = getLatestInvestments(investments ≤ 해당날짜)
차입금   = active 차입금 합계 (고정값)
순현금   = 운전자금 + 운용자금 - 차입금
```

## 표 뷰

| 컬럼 | 설명 |
|------|------|
| 기준일 | |
| 운전자금 | 파란색 |
| 운용자금 | 초록색 |
| 차입금 | 빨간색 |
| 순현금 포지션 | 양수=회색, 음수=빨강 |
| 전일 대비 | ▲빨강(상승) / ▼파랑(하락) / ─ |
| 작성자 | |

## 차트 뷰

Recharts AreaChart 4계열:
- 운전자금 (파랑)
- 운용자금 (초록)
- 차입금 (빨강)
- 순현금 (보라, 점선)

## 딥링크

```
/history/셀바스에이아이/2026-01-01/2026-03-31
→ 날짜 범위 계산 후 가장 가까운 Period 탭 자동 선택
```

## 변경 포인트

- **기간 탭 추가**: `PERIODS` 배열에 항목 추가
- **차트 계열 추가**: AreaChart에 `<Area>` 추가
- **전일 대비 색상**: ▲ 빨강 / ▼ 파랑 (한국 주식 관례) → 색상 교체 시 className 수정
