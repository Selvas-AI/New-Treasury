## 3. 다음 세션 진행 규약 (사용자 합의)

1. 세션 시작 시, 사용자에게 **현재 남은 한도(%)**를 물어본다.
2. 위 1번 표의 예상 소진율과 남은 한도를 비교해 **이번 세션에 안전하게 끝낼 수 있는 항목(묶음)**을 추천한다.
   - 남은 한도 < 예상 소진 + 여유(검증 +10%p)면 더 가벼운 항목으로 하향 조정.
3. **1개 항목을 완료(빌드 통과 + 커밋·푸시 + 문서 갱신)할 때마다**, 사용자에게
   **남은 한도(%)를 다시 물어보고**, 그 값을 기준으로 다음 항목 진행 여부·선택을 가이드한다.
4. 한도가 빠듯하면 "여기서 마무리" 를 먼저 제안한다 (무리한 착수 금지).
5. 각 항목 완료 시 `docs/improvements_session15.md`(또는 신규 세션 문서)에 AS-IS/TO-BE 기록.
