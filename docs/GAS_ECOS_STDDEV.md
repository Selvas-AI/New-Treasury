# GAS ECOS 표준편차 자동계산

> Code.gs v4 신규 기능 (2026-06-05)  
> 한국은행 ECOS Open API를 통한 FX 연간 표준편차 자동계산

---

## 개요

FX 정책 페이지의 "🔄 자동계산 (ECOS)" 버튼 클릭 시:
1. GAS Web App `?type=fxstddev` 호출
2. ECOS API에서 USD/EUR/JPY/GBP 과거 1년 일별 환율 조회
3. 일별 수익률(변동률) 계산 → 표준 표준편차 → 연환산(×√252)
4. 결과를 `policy_params` 테이블에 저장

---

## 사전 준비

1. **ECOS API 키 발급**: https://ecos.bok.or.kr/api/#/ (무료 회원가입)
2. **GAS 스크립트 속성 등록**:
   - GAS 에디터 → 프로젝트 설정(⚙️) → 스크립트 속성
   - `ECOS_API_KEY` = `{발급받은 인증키}`
3. **GAS 재배포**: 새 버전으로 배포

---

## ECOS API 호출 형식

```
https://ecos.bok.or.kr/api/StatisticSearch/{apiKey}/json/kr/1/500/731Y001/D/{from}/{to}/{itemCode}
```

| 파라미터 | 값 | 설명 |
|---|---|---|
| 통계표코드 | `731Y001` | 주요국 통화의 대원화환율 |
| **주기** | **`D`** | 일별 (DD가 아님 — 명세서 확인) |
| 검색일자 형식 | `YYYYMMDD` | 예: 20250605 |

### 통화별 itemCode (731Y001 기준)

| 통화 | itemCode | 항목명 |
|---|---|---|
| USD | `0000001` | 원/미국달러(매매기준율) |
| EUR | `0000003` | 원/유로 |
| JPY | `0000002` | 원/일본엔(100엔) |
| GBP | **`0000012`** | 원/영국파운드 (**0000004는 독일마르크 — 오류**) |

> ⚠️ `StatisticItemList`로 확인한 실제 코드. 직관적 추정(0000001~4) 시 GBP 오류 발생.

---

## 표준편차 계산 공식

```
일별 수익률 = (오늘 환율 - 전일 환율) / 전일 환율
일별 표준편차 = STDEV(일별 수익률 배열)   # 표본 표준편차
연간 표준편차 = 일별 표준편차 × √252      # 연환산 (252 영업일)
```

## 진단 함수

GAS 에디터에서 직접 실행 가능:

| 함수 | 용도 |
|---|---|
| `testFxStdDev()` | 4개 통화 표준편차 계산 결과 로그 출력 |
| `testEcosItemList()` | 731Y001 통계의 전체 53개 항목코드 출력 |
| `testEcosUrlVariants()` | 다양한 URL 형식 시도 (디버깅용) |

---

## 주의사항

- **Rate limit**: ECOS API 3분간 300회 호출 제한 → 통화 간 300ms 대기 포함
- **당일 데이터**: ECOS는 영업일 기준 전일까지 데이터 제공 (T+1)
- **GBP 코드**: `0000004` ≠ GBP (독일마르크 코드), 반드시 `0000012` 사용
- **주기코드**: `DD`는 ECOS에 없는 코드, 반드시 `D` 사용

## 오류 코드

| 코드 | 의미 | 대처 |
|---|---|---|
| `ERROR-100` | 필수값 누락 | 주기코드 `D` 확인, itemCode 확인 |
| `ERROR-602` | Rate limit (3분 300회 초과) | 30분 대기 후 재시도 |
| ECOS_API_KEY 미설정 | GAS 스크립트 속성 누락 | GAS 프로젝트 설정에서 키 등록 |
